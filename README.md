# HR Analytics Dashboard – Power BI Project

## 📌 Project Overview

This project is an HR Analytics Dashboard built using Microsoft Power BI. The dashboard helps analyze employee data and provides meaningful insights related to workforce demographics, salary distribution, employee joining trends, education qualifications, and leave balance.

The main goal of this project is to transform raw HR data into interactive visual reports that support better decision-making for HR teams and management.

---

## 🚀 Features

* Employee demographic analysis
* Gender distribution insights
* Salary analysis by job role
* Employee age analysis
* Education qualification breakdown
* Employee joining trend analysis
* Leave balance monitoring
* Interactive filters and slicers
* Clean and user-friendly dashboard design

---

## 🛠️ Tools & Technologies Used

* **Microsoft Power BI**
* **Power Query** for data cleaning and transformation
* **DAX (Data Analysis Expressions)** for calculated measures
* **Excel Dataset** as the data source

---

## 📂 Dataset Information

The dataset contains HR-related employee information such as:

* Employee ID
* Employee Name
* Gender
* Education Qualification
* Date of Joining
* Job Title
* Salary
* Age
* Leave Balance

---

## 📈 Sample DAX Measures Used

### Total Employees

```DAX
Total Employees = COUNT('HR Data'[Emp ID])
```

### Average Salary

```DAX
Average Salary = AVERAGE('HR Data'[Salary])
```

### Total Leave Balance

```DAX
Total Leave Balance = SUM('HR Data'[Leave Balance])
```

### Average Age

```DAX
Average Age = AVERAGE('HR Data'[Age])
```
